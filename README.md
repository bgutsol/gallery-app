## Installation (development)

1. Clone the repository:
    ```
    $ git clone https://github.com/bgutsol/gallery-app.git
    ```
2. Install dependencies:
    ```
    $ yarn
    ```
    or
    ```
    $ npm install
    ```
3. Run dev-server:
    ```
    $ yarn start
    ```
    or
    ```
    $ npm start
    ```
4. Open http://localhost:3000/
